# CANMOVES Excel Input Generator

This notebook generates a CANMOVES Excel input file with 9 sheets using two JSON configuration files, two CSV files, and user-defined parameters.

The generated Excel workbook includes the following sheets:

```text
Traffic
Region
Fleet
FleetAge
PHEV
Electric
ColdStart
Evaporative
LinkClass
```

## Notebook Structure

The notebook is organized into separate cells:

```text
Cell 1: Import libraries
Cell 2: Define parameters
Cell 3: Read JSON files
Cell 4: Read CSV files
Cell 5-13: Generate individual Excel sheets
Cell 14: Export all sheets into one Excel file
```

## Required Input Files

The code uses the following input files:

```text
canmovesScaleDefinition.json
canmovesRunSpecConfig.json
Fleet CSV file
FleetAge CSV file
```

The JSON files provide model configuration values such as fleet classification, selected year, month, temperature, electrical grid settings, PHEV settings, cold-start settings, evaporative emission settings, and link-class definitions.

The CSV files are used directly to create the `Fleet` and `FleetAge` sheets.

## Parameters

The main parameters are defined in the parameters cell, including file paths such as:

```python
CANMOVES_SCALE_DEFINITION_PATH
CANMOVES_RUN_SPEC_CONFIG_PATH
Fleet_CSV_PATH
FleetAge_CSV_PATH
EMME_Path
OUTPUT_EXCEL_PATH
```

These parameters can be edited depending on the project folder, input files, and desired output location.

## Reading JSON Files

The JSON-reading functions load the two CANMOVES JSON files as Python dictionaries.

```python
json_inputs = read_canmoves_json_files(
    scale_definition_path=CANMOVES_SCALE_DEFINITION_PATH,
    run_spec_config_path=CANMOVES_RUN_SPEC_CONFIG_PATH
)

scale_definition = json_inputs["scale_definition"]
run_spec_config = json_inputs["run_spec_config"]
```

This allows the sheet-generation functions to access values directly, for example:

```python
scale_definition["fleetClassification"]
run_spec_config["domainInfo"]["selectedYear"]
run_spec_config["electricalGrid"]["powerShares"]
```

## Reading CSV Files

The CSV-reading functions load the `Fleet` and `FleetAge` files as dataframes.

The CSV files are read with `header=None`, so the original CSV header row is kept inside the dataframe. This allows all sheets to be exported using `header=False`.

```python
csv_inputs = read_canmoves_csv_files({
    "Fleet": Fleet_CSV_PATH,
    "FleetAge": FleetAge_CSV_PATH
})

Fleet = csv_inputs["Fleet"]
FleetAge = csv_inputs["FleetAge"]
```

## Sheet Generation Functions

Each Excel sheet is generated by a separate function. Each function returns one pandas dataframe.

```python
Traffic = generate_traffic_sheet(scale_definition, run_spec_config, EMME_Path)
Region = generate_region_sheet(run_spec_config)
Fleet_sheet = generate_fleet_sheet(Fleet)
FleetAge_sheet = generate_fleetage_sheet(FleetAge)
PHEV = generate_phev_sheet(run_spec_config)
Electric = generate_electric_sheet(run_spec_config)
ColdStart = generate_coldstart_sheet(run_spec_config)
Evaporative = generate_evaporative_sheet(run_spec_config)
LinkClass = generate_linkclass_sheet(run_spec_config)
```

The `Fleet` and `FleetAge` sheets are direct copies of the corresponding CSV dataframes.

The other sheets are generated from values stored in the JSON configuration files and parameters.

## Exporting the Excel File

All generated dataframes are collected into one dictionary and exported to a single Excel workbook.

```python
sheets = {
    "Traffic": Traffic,
    "Region": Region,
    "Fleet": Fleet_sheet,
    "FleetAge": FleetAge_sheet,
    "PHEV": PHEV,
    "Electric": Electric,
    "ColdStart": ColdStart,
    "Evaporative": Evaporative,
    "LinkClass": LinkClass
}

saved_file = export_canmoves_excel(
    sheets=sheets,
    output_excel_path=OUTPUT_EXCEL_PATH
)

print(f"Excel file saved to: {saved_file}")
```

All sheets are exported using:

```python
index=False
header=False
```

This is because each dataframe is already formatted to match the desired Excel layout.

## Output

The final output is one Excel file containing all 9 CANMOVES input sheets.

Example output path:

```python
/content/CANMOVES_Batch_Input.xlsx
```
